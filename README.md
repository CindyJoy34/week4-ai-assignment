## Q2- Difference between structured and unstructured data
Structured data had a fixed format/schema and fits nearly into rows and columns eg. Names and dates while unstructured data does not have a fixed format/shcema eg. Images and audio.
## Q3- What is schema?
This is a structured framework or blueprint that organizes information, concepts, or data.
